# NGO Awareness Website 🌱

A simple and responsive NGO awareness webpage developed using **HTML** and **CSS**.

The purpose of this project is to spread awareness about an NGO's mission, ongoing initiatives, and social impact while encouraging users to volunteer and support the cause.

---

## 🚀 Features

- Responsive landing page
- About the NGO section
- Ongoing projects section
- Social impact information
- Call-to-Action section
- Clean and user-friendly UI

---

## 🛠️ Technologies Used

- HTML5
- CSS3

---

```
├── index.html

```

## 🎯 Purpose

This project was created as part of an internship task to practice frontend web development while designing a webpage that promotes awareness about social initiatives.
